Foodie Finder App

User Research Report

For my 203 User Experience Design class I did a long project centered around a new app idea. I chose to make an app centered around dining in the Harrisonburg, VA area. I acted as a UX designer in creating a problem statement, conducting research to verify the problem, and created specifications for the end app. This PDF documents this process.
